# PlantMDCS
  PlantMDCS is a code-free, modular graphical toolkit for rapid plant multi-omics database construction with low technical barriers. A standalone Windows executable, it auto-generates a graphical backend and web-based frontend, supporting full-process data operation without coding. It is compatible with major plant multi-omics data types and integrates comprehensive functional modules for plant research analysis needs.
  The Windows version of PlantMDCS1.0 have been completed. You can download it from "<a href="https://github.com/Chen-Chen-888/PlantMDCS/releases/tag/PlantMDCS">Releases</a>".
